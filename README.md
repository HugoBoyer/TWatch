# Movie Catalog Portfolio

A React + TypeScript + Vite project showcasing a catalog of movies and series using the TMDb API.  
This project is for personal portfolio purposes and demonstrates React, API fetching, and UI animations.

---

## Features

- Browse popular movies and series
- View detailed movie pages with actors, genres, and description
- Search and filter movies
- Smooth UI animations using Framer Motion
- Responsive design for desktop and mobile

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Animations:** Framer Motion
- **Styling:** Tailwind CSS (or your preferred CSS framework)
- **API:** TMDb API (free for personal use)
